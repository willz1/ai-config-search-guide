# Contributing

Contributions are welcome! Here's how you can help:

## Adding a New Search Trick

1. Fork this repository
2. Add your search trick following this format:

```markdown
| `config-path` | `search-query` | Brief description |
```

3. Ensure the search query actually works on GitHub
4. Submit a pull request

## Guidelines

- Keep descriptions concise
- Test your search queries before submitting
- Follow the existing table format
- One trick per row

## Questions?

Open an issue if you have questions or suggestions.
